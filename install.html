📋 How to install

🤖 Android (Chrome / Edge)

  1. Tap the menu (⋮) in the top-right corner.
  2. Tap "Install app" or "Add to Home Screen".
  3. Confirm by tapping "Install".
  4. The app appears on your home screen with the school logo.

✓ Push notifications work after install.

🍎 iPhone / iPad (Safari)

  1. Open this page in Safari (not Chrome).
  2. Tap the Share button ⬆️ at the bottom.
  3. Scroll down and tap "Add to Home Screen".
  4. Tap "Add" in the top-right corner.

⚠️ iOS only supports push if the app is added to Home Screen.

🪟 Windows / Mac (Edge / Chrome)

  1. Look for the install icon (⊕) in the address bar.
  2. Click it and choose "Install".
  3. The app opens in its own window, like a real program.
  4. Pin it to your taskbar / dock for quick access.

✓ Push notifications work in Edge & Chrome on desktop.

❓ Why a banner keeps appearing

Every page on the platform shows the install banner until the app is installed. This is intentional — we want every parent and teacher to actually install it so they receive push notifications for messages, broadcasts and polls. You can dismiss it once but it will return after a couple of days (configurable).
